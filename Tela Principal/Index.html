O que é o Pindorama?

O Pindorama é um portal educacional com o objetivo de disponibilizar gratuitamente todo o tipo de conhecimento para o seu aprimoramento tanto como aluno quanto como profissional.

Estaremos disponibilizando vários tipos de materiais, como videoaulas, conteúdo em PDFs, ferramentas para acompanhar seu progresso, entre outros.
